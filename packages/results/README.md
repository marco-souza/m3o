# Results (`@m3o/results`)

A JavaScript and TypeScript library to add Result types to your code.

This is compatible with Node.js, Deno and Bun via [`jsr.io`](httos://jsr.io).

## Usage

```sh
# run deno form jsr
deno install jsr:@m3o/results

# with node
npx jsr add @m3o/results

# with bun
bux jsr add @m3o/results
```
